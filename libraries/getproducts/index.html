    Get Product List

    GET Get list of products.

    https://api.vfs.velma.com/v1-product/products

    Params¶
    KEY VALUE
    productId 202

    This method accepts the following arguments.

    Param Type Default Description
    token String N/A An authority token providing at least LIB:30 access.
    jobIds Array N/A A list of jobIds to finalize.
    disposition String N/A The disposition to set on the finalized jobs. One of: approved, declined
    comments String None Optional: Custom comments from the person that finalized the request(s).

    Headers¶

    KEY VALUE DESCRIPTION
    Authorization jwt {{Token}} generated by Velma
    x-api-key 49ep6U3OEH2UR56Dg4BNp6qKxNCnKeDP3l29cyZg provided following on-boarding
    Content-Type application/json
